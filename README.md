## Pokemon API Challenge - Next.js / TypeScript Version

### Description
A single page Pokemon application built with **Next.js**, **TypeScript**, **Tailwind CSS**, and the **PokeAPI**.
Users can search for Pokemon by name or Pokedex number, get a random Pokemon, view the normal and shiny images, see the location, view all abilities, all moves, the evolution path, and save/remove the Pokemon to/from their favorites list.
